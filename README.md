# Deep Learning Projects

Built from scratch while learning deep learning.

## Projects

### 1. Neural Network from Scratch (NumPy)
- Implemented forward pass, backpropagation, gradient descent
- Solved XOR problem without any ML framework
- Key learning: understood what Keras does underneath

### 2. Titanic Survival (Keras)
- Binary classification with Dense layers
- Used Dropout, Early Stopping, validation monitoring
- Result: 81.56% accuracy

### 3. MNIST Digit Recognition (CNN)
- Convolutional Neural Network from scratch
- Result: 99.17% accuracy

### 4. CIFAR-10 Image Classification (Transfer Learning)
- Used MobileNetV2 pretrained on ImageNet
- Phase 1: 81% → Phase 2 fine tuning: 82.73%

### 5. IMDB Sentiment Analysis (LSTM)
- Sequential text classification
- Result: 86.24% accuracy

## Stack
Python, TensorFlow, Keras, NumPy, Pandas, Scikit-learn
